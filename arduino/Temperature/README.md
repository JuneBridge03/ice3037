# Arduino &#8594; Temperature

* 센서 형번 _MLX90614_ 
* 라이브러리 by _Adafruit_
* 방사율 고려한 캘리브레이션 필요함
    -> 최근 10개의 값의 평균을 사용해서 제어하는 방안 생각 중, 알고리즘 자체는 동일. 데이터 처리만 요
    -> 11주차 실습 시간에 해결할 예정
* 현재 모델은 체온과 주변온도 측정 가능. 체온 측정 시에 피부와 접촉 시 온도가 상당히 높게 측정되는 현상 발견됨.
    * 해결 방안
        1. 우선 체온 평균치 사용으로 해결.
        2. 신태하 조장이 3d 프린팅으로 외부 프레임으로 생성, 접촉 불가능하게 함.
