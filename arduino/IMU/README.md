# Arduino &#8594; IMU

## 3-axes
* _ADXL345_ 사용
* 중력 제거 및 SO3 추정이 불가능하여 중단
## 6-axes
* _MPU6050_ 사용
* _Electronic Cats_ 의 라이브러리 사용
* _Zero_ 프로젝트를 수행하여 캘리브레이션 가능
* _DMP6_ 프로젝트에서 teapot 시각화 확인 가능
### _Fusion_ 프로젝트
* 복합적인 물리량으로부터 미동 판단 목적
* pre-calib 값이 있을 경우 건너뜀
## 9-axes
* _MPU9250_ 사용
* 지자기 축으로 장기적인 탈조 억제 가능하나 불필요
* 값이 튀어서 보류
